## 🌌 Overview

The **NeuroByte Society** is the premier nexus for students passionate about the intersection of life sciences and computational intelligence. We build, research, and innovate in the realms of Artificial Intelligence, Bioinformatics, Biotechnology, and modern Healthcare solutions.

This repository serves as our digital headquarters, beautifully crafted with modern web technologies, fluid animations, and a responsive design tailored for the future.

## ✨ Features

- ⚡️ **Blazing Fast**: Powered by **Vite** and **Bun** for instant hot-module replacement and rapid builds.
- 🛣️ **Type-Safe Routing**: Built with **TanStack Start & Router** for flawless and safe navigation.
- 🎨 **Fabulous Design**: Crafted with **Tailwind CSS**, featuring dark mode, glassmorphism, and dynamic micro-animations.
- 🧩 **Radix Primitives**: Fully accessible, unstyled UI components brought to life with custom styling.
- 📱 **Fully Responsive**: Adapts seamlessly from mobile devices to ultrawide desktop monitors.

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Make sure you have [Bun](https://bun.sh/) installed on your system. Alternatively, you can use `npm` or `pnpm`.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/adilsukumar/neuro-byte-nexus.git
   cd neuro-byte-nexus
   ```

2. **Install dependencies**
   ```bash
   bun install
   ```

3. **Start the development server**
   ```bash
   bun run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to see the application running.

## 🏗️ Build for Production

To build the application for production, run:

```bash
bun run build
```

This will generate the optimized client and server builds in the `dist` directory.
